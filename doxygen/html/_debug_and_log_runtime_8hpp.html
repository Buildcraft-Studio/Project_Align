class  Engine::Runtime::DebugAndLogRuntime
 A runtime module in its own right (implements IRuntimeModule, so it gets a WatchDog'd table entry like everything else), but it also doubles as the read/write entry point into the table for humans/tools: it can dump every runtime's status (read) and can flip a runtime's status flag by hand (set), e.g. to mark something INACTIVE or force it back to STARTED.
