Detailed Description

Loads and registers every runtime module BEFORE management is handed off to RuntimeManger. Per RuntimeManger's own doc note, RuntimeManger itself is assumed to be runtime ID 0; Bootstrap registers everything else on top of that.

Each module Bootstrap registers gets createRuntime()'d (RuntimeManger::setRuntime()) with status STARTED (14) - loaded, but not yet WatchDog-checked. Bootstrap owns the module instances so their lifetime outlives the RuntimeManger that only holds raw pointers to them.
